- Check that you opened the correct file
- If the file is inside a ZIP folder, extract it first

### Windows blocks the file

- Right-click the file
- Select **Properties**
- If you see an **Unblock** option, turn it on
- Select **Apply**
- Open the file again

### The window looks small

- Use the maximize button in the top-right corner
- Check your screen scaling in Windows
- Set your display to the recommended resolution if needed

### The app closes right away

- Restart your computer
- Open the app again
